## Break-Shop


e 커머스를 구현한다 가정. 

그 백엔드 파트를 단독으로 구현해보며, 그 과정과 트러블 슈팅 과정을 담는다.

## 중점사항 
- CI/CD 구축을 통한 자동/무중단 배포 구현

- Redis를 통한 세션 인증방식 도입

- 스프링에서의 단위 테스트 구현
